# Archived documentation

This directory holds **historical snapshots** that are kept for audit trail and
regression comparison only. They are **not** the current source of truth.

## June 2026 skills user-story gap reports

Files named `skills-user-stories-gap-report-2026-06-*` in the parent [`docs/`](../)
directory (JSON and Markdown) were produced during the June 2026 MCP parity audit
cycle. They reference the **pre–10-tool**
hosted MCP surface — for example `loxtep_data_products`, `loxtep_ontology`,
`loxtep_catalog`, and other legacy facade names that were consolidated into the
current ten tools:

`loxtep_session`, `loxtep_connect`, `loxtep_workspace`, `loxtep_build`,
`loxtep_define`, `loxtep_meaning`, `loxtep_review`, `loxtep_query`,
`loxtep_observe`, `loxtep_context`.

When checking current MCP behavior, use [`AGENTS.md`](../../AGENTS.md) and
[`docs/skills-user-stories.md`](../skills-user-stories.md) instead of these
archived reports.
